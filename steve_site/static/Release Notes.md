## v1.0.0 *(2026-06-21)*

- 🎉 **Release**: 正式上线

核心特性: 

1. **角色访问控制 (RBAC) 系统**: 多用户角色与多级博客权限，根据用户身份动态显隐并校验操作
2. **云对象存储与动静分离架构**: 媒体资产接入 Cloudflare R2 对象存储，并借助 Cloudflare CDN 缓存与分发；静态资源与源服务器解耦，带来丝滑的加载体验

> Cloudflare 在某些地区的网络连通性可能不佳
